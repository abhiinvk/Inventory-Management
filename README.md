# Inventory Management Dashboard (Excel)

## Project Overview

This project is an Inventory Management Dashboard built in Microsoft Excel to monitor inventory levels, stock movements, and product performance. The dashboard helps businesses track current stock, identify low-stock items, analyze inventory value, and monitor sales performance through interactive reports and visualizations.

## Dataset

The project uses three datasets:

* Products (500 records)
* Purchases (5,000 records)
* Sales (7,000 records)

## Features

* Product inventory tracking
* Purchase and sales analysis
* Current stock calculation
* Inventory value calculation
* Low stock alerts
* KPI reporting
* Pivot Tables and Pivot Charts
* Interactive dashboard with slicers

## KPIs

The dashboard includes the following key performance indicators:

* Total Products
* Total Inventory Value
* Low Stock Items
* Total Units Sold

## Excel Skills Used

* VLOOKUP
* SUMIFS
* IF
* COUNTIF
* Pivot Tables
* Pivot Charts
* Sorting and Filtering
* Slicers
* Dashboard Design
* Data Validation

## Dashboard Components

### Charts

* Inventory Value by Category
* Current Stock by Category
* Top 10 Products by Units Sold
* Stock Status Distribution

### Tables

* Low Stock Items
* Top Products by Inventory Value

### Filters

* Category
* Stock Status

## Business Insights

The dashboard enables users to:

* Monitor inventory levels in real time
* Identify products that require reordering
* Analyze inventory value across categories
* Track top-selling products
* Support inventory planning and business decisions

## Project Outcome

Developed a professional Inventory Management Dashboard in Excel that demonstrates inventory tracking, reporting, data analysis, and dashboard development skills commonly used in MIS Executive, Operations Executive, and Data Analyst roles.
